## CLI Examples

```bash
# Installation
pip install llmforge          # Python
cargo install llm-forge        # Rust
npm install -g llm-forge       # Node.js
brew install llm-forge         # macOS (future)

# Usage
llm-forge status               # Check worker pool
llm-forge spawn --model=sonnet --count=3
llm-forge dashboard            # Launch TUI
llm-forge optimize             # Run cost optimization
llm-forge metrics              # Show performance metrics

# Short alias (optional)
forge status
forge spawn -m sonnet -n 3
forge dashboard
```

## Competitive Differentiation

### vs. Manual Agent Spawning
- **Problem**: Manually launching and managing 5+ tmux sessions is tedious
- **LLM-Forge**: Single dashboard, one-key spawning, automatic health checks

### vs. Direct API Usage
- **Problem**: Pay-per-token gets expensive fast, hard to optimize
- **LLM-Forge**: Intelligent routing to subscriptions first, API as overflow

### vs. Task Queues (Celery, BullMQ)
- **Problem**: Generic task queues don't understand AI model costs/capabilities
- **LLM-Forge**: Task value scoring, model-aware routing, cost forecasting

### vs. Kubernetes/Docker
- **Problem**: Heavy infrastructure, not designed for interactive AI agents
- **LLM-Forge**: Lightweight, agent-specific, developer-friendly

## FAQ for Branding

**Q: Why "LLM-Forge" and not just "Forge"?**
A: The "LLM-" prefix makes the purpose immediately clear and avoids conflicts with the hundreds of other "forge" tools in software development.

**Q: How do you pronounce it?**
A: "L-L-M Forge" (spell out L-L-M, then "forge" like a blacksmith's forge)

**Q: Can I use it for my project?**
A: The name "LLM-Forge" will be trademarked for AI orchestration software. You're welcome to use the tool, but please don't create competing projects with the same name.

**Q: Is there a shorter alias?**
A: Yes, the CLI supports both `llm-forge` and `forge` as aliases.

**Q: Will you rebrand?**
A: No. The availability analysis shows strong availability across domains, packages, and social media. We're confident in "LLM-Forge" as the long-term brand.
