## How to Add Screenshots

1. Run FORGE in a terminal
2. Use terminal screenshot tools:
   - macOS: `Cmd+Shift+4` (select area)
   - Linux: `gnome-screenshot` or `scrot`
   - Windows: `Win+Shift+S` (Snipping Tool)
3. Save to this directory with descriptive names:
   - `3-column-layout.png`
   - `cost-analytics.png`
   - `subscription-panel.png`
   - `chat-interface.png`
   - `theme-default.png`
   - `theme-dark.png`
   - `theme-light.png`
   - `theme-cyberpunk.png`

4. Update documentation to reference screenshots
