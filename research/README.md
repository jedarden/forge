# Control Panel - Intelligent Worker Management System

**Formerly**: Pool Optimizer
**New Name**: Control Panel (better reflects complete feature set)

## What is the Control Panel?

An intelligent **TUI (Text User Interface) dashboard** for managing distributed coding agents across multiple workspaces, models, and subscriptions.

### Core Capabilities

1. **Worker Pool Management** - Spawn/kill workers, health monitoring, auto-recovery
2. **Cost Optimization** - 60-98% savings through intelligent model assignment
3. **Task Distribution** - Value-based scoring and dependency-aware scheduling
4. **Real-Time Dashboard** - Responsive TUI supporting 80×24 to 240×50 terminals
